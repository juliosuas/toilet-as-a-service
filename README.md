# Toilet as a Service

Una sátira de producto demasiado plausible: acceso a baños públicos financiado por anuncios imposibles de omitir.

## La experiencia

- Landing editorial completamente responsive.
- Demo funcional: anuncio de 12 segundos, verificación y pase de acceso.
- Planes para usuarios y operadores, métricas, testimonios y letra pequeña.
- Tarjeta social Open Graph diseñada para el proyecto.

## Desarrollo

Requiere Node.js 22.13 o posterior.

```bash
npm install
npm run dev
```

Validación de producción:

```bash
npm test
```

## Aviso sanitario

TaaS es una pieza satírica. Ninguna vejiga fue monetizada durante su desarrollo.
